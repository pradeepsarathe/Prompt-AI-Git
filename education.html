Learning Paths

Pick your direction

Curated sequences depending on what you want to achieve with AI.

Path 01

Use AI Productively

For professionals who want to integrate AI tools into their daily workflow — no coding required.

  • Start with AI for Everyone (Andrew Ng)
  • Learn Google AI Essentials for workplace tools
  • Master prompt engineering via Learn Prompting
  • Pick 1–2 tools: Claude, ChatGPT, Perplexity
  • Follow The Batch newsletter weekly
Path 02

Build AI Applications

For developers who want to build products using AI APIs, fine-tuning, and deployment pipelines.

  • Complete ML Specialization (Andrew Ng)
  • Do Practical Deep Learning (fast.ai)
  • Study the HuggingFace NLP Course
  • Build with LangChain & vector databases
  • Deploy: LLMs in Production (Databricks)
Path 03

Research AI Systems

For those aiming to understand, replicate, or advance the state of the art in machine learning.

  • Study the Deep Learning Specialization rigorously
  • Complete Stanford CS224N or CS231N
  • Rebuild GPT from scratch — Karpathy's Zero to Hero
  • Read arXiv papers daily via PromptAI Papers
  • Follow The Gradient and Distill.pub
