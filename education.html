Use AI productively

Beginner

For professionals who want to integrate AI tools into their daily workflow — no coding required.

Time ~20 hrs Steps 5 Cost Free
  1. Start with AI for Everyone (Andrew Ng)
  2. Learn Google AI Essentials for workplace tools
  3. Master prompt engineering via Learn Prompting
  4. Pick 1–2 tools: Claude, ChatGPT, Perplexity
  5. Follow The Batch newsletter weekly

Build AI applications

Intermediate

For developers who want to build products using AI APIs, fine-tuning, and deployment pipelines.

Time ~60 hrs Steps 5 Code Python
  1. Complete ML Specialization (Andrew Ng)
  2. Do Practical Deep Learning (fast.ai)
  3. Study the HuggingFace NLP Course
  4. Build with LangChain & vector databases
  5. Deploy: LLMs in Production (Databricks)

Research AI systems

Advanced

For those aiming to understand, replicate, or advance the state of the art in machine learning.

Time ~120 hrs Steps 5 Math Heavy
  1. Study the Deep Learning Specialization rigorously
  2. Complete Stanford CS224N or CS231N
  3. Rebuild GPT from scratch — Karpathy's Zero to Hero
  4. Read arXiv papers daily via PromptAI Research
  5. Follow The Gradient and Distill.pub
